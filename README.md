# 新能源汽车数据分析系统 python975

## 项目概述

新能源汽车数据分析系统是基于Python开发的一套数据管理与分析工具。它旨在帮助用户和管理员高效处理和查看新能源汽车相关的数据。

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 前端：Vue

- 系统首页管理
- 个人中心管理
- 用户管理

- Python 环境配置
- MySQL 数据库环境
- Vue 前端运行环境

## 部署步骤

1. 环境准备：确保Python、MySQL、Vue等相关环境安装配置完成。
2. 数据库搭建：根据提供的数据库结构文档，在MySQL中创建相应的数据库和表。
3. 后端部署：将后端代码部署到服务器上，并配置好相关环境变量。
4. 前端部署：将Vue项目构建后部署到Web服务器。
